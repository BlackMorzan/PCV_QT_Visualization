Detailed Description

Jest to główna klasa obsługująca komunikację między UI, a resztą programu.

Klasa zawiera metody związane z UI, jak obsługa przycisków, przesyłanie watrości do wyświetlania na wykresach i rozpoczyna nowy wątek
